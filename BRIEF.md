# Daily Focus — Project Brief

## Product
A simple single-page website named **Daily Focus** where the user sets one main goal for the day.

## Requirements
- Page title: "Daily Focus"
- Text input for the user’s main goal for today
- Button labeled "Save Goal"
- On click, the goal is shown below the button
- The goal persists after page refresh (browser storage acceptable)

## Technical constraints
- **HTML, CSS, and JavaScript** in the browser — no UI frameworks (React, Vue, etc.)
- Optional **TypeScript compile** (`npm run build`) for typed components and the page assembler; shipped assets remain plain JS modules
- Design: clean, centered layout

## Out of scope (v1)
- Backend server, accounts, or cloud sync
- Multiple goals or history
